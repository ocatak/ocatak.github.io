For more information about my research, you can also visit my GitHub and Google scholar profiles. I am interested in Data Privacy and Cyber Security for machine learning and deep learning, covering the theoretical, applicative, and computational aspects. My primary research interests lie broadly in deep learning.

Collaborations on 5G and Beyond Cyber Security

My research outputs and implementations can be found on CyberSecurity of 5G/6G Networks page. If you are interested in collaborating on 5G and beyond cyber security research, please do not hesitate to contact me.

  • Next generation wireless communication (5G, 6G, 7G etc.) security
  • Adversarial Machine Learning
  • Homomorphic Encryption based Data Privacy
  • Machine learning methods in Cyber Security (Malware analysis, network flows)
  • Uncertainty methods in deep learning
